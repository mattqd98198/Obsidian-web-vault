## 🎯 项目概述

### 核心功能
web-scout-mcp 是一个**MCP (Model Context Protocol) 服务器**，主要提供：

1. **🔍 DuckDuckGo 搜索**
   - 快速、注重隐私的网络搜索
   - 不依赖Google API
   - 免费使用

2. **📄 内容提取**
   - 从网页提取干净、易读的文本
   - 自动移除脚本、样式、导航等无关内容
   - 支持单个或批量URL处理

3. **🚀 并行处理**
   - 同时处理多个URL
   - 提高效率

4. **💾 内存优化**
   - 智能内存管理
   - 防止应用崩溃

5. **⏱️ 速率限制**
   - 智能请求节流
   - 避免API阻塞

6. **🛡️ 错误处理**
   - 健壮的错误处理机制
   - 确保可靠运行

**方式1: 全局安装**
```bash

```

**方式2: 项目内安装**
```bash

```

**方式3: npx直接运行（推荐）**
```bash
npx -y @pinkpixel/web-scout-mcp
```

**方式4: 通过Smithery自动安装**
```bash
npx -y @smithery/cli install @pinkpixel-dev/web-scout-mcp --client claude
```

### 配置方式

添加到Claude Desktop的MCP配置 (`config.json`):

```json
{
  "mcpServers": {
    "web-scout": {
      "command": "npx",
      "args": [
        "-y",
        "@pinkpixel/web-scout-mcp@latest"
      ]
    }
  }
}
```

## 📊 功能分析

### MCP工具列表

#### 1. `web_search` - 网络搜索
**输入**:
- `query` (string): 搜索关键词
- `maxResults` (number, 可选): 最大结果数（默认10）

**示例**:
```json
{
  "query": "latest advancements in AI",
  "maxResults": 5
}
```

**输出**: 包含标题、URL、摘要的结构化搜索结果

#### 2. `fetch_webpage` - 网页内容提取
**输入**:
- `url`: 单个URL字符串或URL数组

**示例**（单个URL）:
```json
{
  "url": "https://example.com/article"
}
```

**示例**（多个URL）:
```json
{
  "url": [
    "https://example.com/article1",
    "https://example.com/article2"
  ]
}
```

**输出**: 从指定URL提取的纯文本内容

## ⚠️ 风险和限制

### 1. 技术风险
- ⚠️ **依赖DuckDuckGo**: 如果DDG服务变化，可能受影响
- ⚠️ **内容提取准确性**: 可能无法完美提取所有网站
- ⚠️ **JavaScript渲染**: 不支持动态加载的内容（SPA）
- ⚠️ **反爬虫**: 某些网站可能阻止爬取

### 2. 功能限制
- ⚠️ **搜索质量**: DuckDuckGo搜索结果可能不如Google
- ⚠️ **无图片/视频**: 只提取文本，不提取媒体文件
- ⚠️ **无JavaScript执行**: 无法获取动态生成的内容
- ⚠️ **依赖Node.js**: 需要Node.js环境

### 3. 维护风险
- ⚠️ **社区规模**: 个人开发者项目，可能维护不稳定
- ⚠️ **文档**: 缺少详细的API文档
- ⚠️ **测试**: 无明确的测试覆盖报告
