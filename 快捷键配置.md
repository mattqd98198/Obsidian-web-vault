# ⌨️ Tag Group Manager 快捷键配置指南

## 🔧 核心快捷键

### 基础操作
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| `Ctrl+K` | 打开标签搜索 | 快速搜索并插入标签 |
| `Ctrl+N` | 新建标签组 | 快速创建新的标签组 |
| `Ctrl+E` | 编辑模式 | 切换编辑/查看模式 |
| `Ctrl+F` | 聚焦搜索框 | 快速定位到搜索框 |

### 标签操作
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| `Ctrl+Click` | 插入标签 | 快速插入选中标签 |
| `Ctrl+D` | 删除标签 | 在编辑模式下删除标签 |
| `Ctrl+R` | 重命名标签 | 在编辑模式下重命名标签 |
| `Ctrl+S` | 保存更改 | 保存当前所有更改 |

### 集合切换
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| `Alt+1` | 切换到技术集 | 快速切换到技术相关集合 |
| `Alt+2` | 切换到内容集 | 快速切换到内容管理集合 |
| `Alt+3` | 切换到开发集 | 快速切换到开发工具集合 |
| `Alt+0` | 显示全部 | 显示所有标签组 |

### 高级操作
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| `Ctrl+Shift+T` | 批量标签操作 | 打开批量操作面板 |
| `Ctrl+Shift+I` | 导入数据 | 导入标签组数据 |
| `Ctrl+Shift+E` | 导出数据 | 导出标签组数据 |
| `Ctrl+Shift+A` | 打开设置 | 打开插件设置面板 |

## 🎮 自定义快捷键配置

### 在Obsidian中配置
1. 打开 Obsidian 设置
2. 进入 快捷键 选项卡
3. 搜索 "Tag Group Manager"
4. 为每个功能分配快捷键

### 推荐快捷键配置
```json
{
  "tag-group-manager:open-search": "Ctrl+K",
  "tag-group-manager:new-group": "Ctrl+N",
  "tag-group-manager:toggle-edit-mode": "Ctrl+E",
  "tag-group-manager:focus-search": "Ctrl+F",
  "tag-group-manager:insert-selected-tag": "Ctrl+Click",
  "tag-group-manager:delete-tag": "Ctrl+D",
  "tag-group-manager:rename-tag": "Ctrl+R",
  "tag-group-manager:save-changes": "Ctrl+S",
  "tag-group-manager:switch-tech-set": "Alt+1",
  "tag-group-manager:switch-content-set": "Alt+2",
  "tag-group-manager:switch-dev-set": "Alt+3",
  "tag-group-manager:show-all": "Alt+0",
  "tag-group-manager:bulk-operations": "Ctrl+Shift+T",
  "tag-group-manager:import-data": "Ctrl+Shift+I",
  "tag-group-manager:export-data": "Ctrl+Shift+E",
```

## 🖱️ 鼠标操作

### 基础操作
- **单击标签**：插入标签到当前笔记
- **双击标签**：在编辑模式下编辑标签
- **右键标签**：显示标签操作菜单
- **拖拽标签**：重新排序标签（编辑模式下）

### 编辑模式操作
- **拖拽标签组**：重新排列标签组顺序
- **拖拽标签**：在组内重新排序
- **右键标签**：
  - 编辑标签名称
  - 删除标签
  - 复制标签
  - 查看使用统计

### 界面操作
- **点击集合按钮**：切换不同的标签集合
- **点击刷新按钮**：重新加载标签组
- **点击编辑按钮**：切换编辑模式
- **点击搜索框**：快速搜索标签

## 🔍 搜索功能

### 基础搜索
- **实时搜索**：输入时实时显示结果
- **模糊匹配**：支持模糊搜索
- **高亮显示**：搜索结果高亮显示
- **历史记录**：保存搜索历史

### 高级搜索
```javascript
// 搜索语法示例
"React Vue"         // 搜索包含React或Vue的标签
"React AND Vue"     // 搜索同时包含React和Vue的组
"React NOT Vue"     // 搜索包含React但不包含Vue的组
"tag:React"         // 搜索包含React标签的组
"group:tech"        // 搜索tech组内的标签
```

### 搜索快捷键
| 快捷键 | 功能 | 描述 |
|--------|------|------|
| `Ctrl+/` | 切换搜索 | 快速切换搜索框显示状态 |
| `Esc` | 清空搜索 | 清空当前搜索内容 |
| `Enter` | 确认搜索 | 确认当前搜索 |
| `Tab` | 下一个结果 | 切换到下一个搜索结果 |
