# 🌐 Clipper浏览器扩展配置指南

## 安装浏览器扩展

### Chrome浏览器
1. 访问 Chrome Web Store
2. 搜索 "Obsidian Web Clipper"
3. 点击 "添加到Chrome"
4. 确认安装权限

### Firefox浏览器
1. 访问 Firefox Add-ons
2. 搜索 "Obsidian Web Clipper"
3. 点击 "添加到Firefox"
4. 确认安装权限

### Edge浏览器
1. 访问 Microsoft Edge Add-ons
2. 搜索 "Obsidian Web Clipper"
3. 点击 "获取"
4. 确认安装权限

## 扩展设置

### 基础配置
1. 打开扩展选项页面
2. 配置Obsidian vault路径
3. 设置默认剪藏文件夹
4. 选择默认模板

### 高级配置
1. 启用AI功能（可选）
2. 配置自动分类规则
3. 设置内容过滤规则
4. 配置剪藏模式

## 使用指南

### 剪藏方式
1. **一键剪藏**：点击扩展图标，选择剪藏模式
2. **区域剪藏**：选择页面内容区域，右键选择剪藏
3. **全文剪藏**：选择剪藏全文内容
4. **链接保存**：仅保存链接和基本信息

### 快捷键
- `Ctrl+Shift+C`：快速剪藏
- `Ctrl+Shift+V`：从历史中选择
- `Ctrl+Shift+E`：打开扩展选项

### 剪藏模式
1. **全文模式**：保存完整页面内容
2. **摘要模式**：保存标题和摘要
3. **链接模式**：仅保存链接
4. **自定义模式**：根据规则剪藏

## 故障排除

### 常见问题
1. **连接失败**：检查Obsidian是否运行
2. **权限问题**：检查扩展权限设置
3. **路径错误**：确认vault路径正确
4. **模板问题**：检查模板文件是否存在

### 解决方案
1. 重新安装扩展
2. 重启Obsidian
3. 检查网络连接
4. 查看扩展日志
