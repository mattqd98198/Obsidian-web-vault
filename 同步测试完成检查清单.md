## 🚀 自动化配置建议

### Windows计划任务设置
1. **打开任务计划程序**: `taskschd.msc`
2. **创建基本任务**:
   - 名称: `Obsidian GitHub Sync`
   - 触发器: 每天固定时间
   - 操作: 启动 `powershell.exe -ExecutionPolicy Bypass -File "sync.ps1"`

### GitHub Actions配置
可以创建 `.github/workflows/sync.yml` 文件实现CI/CD同步：

```yaml
name: 自动同步
on:
  schedule:
    - cron: '0 22 * * *'  # 每天晚上10点
  workflow_dispatch:

jobs:
  sync:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: 设置Git配置
        run: |
          git config user.name "GitHub Action"
          git config user.email "action@github.com"
      - name: 同步更改
        run: |
          git add .
          git commit -m "自动同步 $(date)"
          git push
```
