# 📋 网页剪藏使用说明

## 文件夹说明

### `01-Inbox/网页剪藏/`
- 存放所有通过Clipper收集的网页内容
- 使用文件命名格式：`YYYY-MM-DD_网站名称_文章标题.md`
- 支持多种剪藏模式和模板

### `01-Inbox/未分类/`
- 临时存放未分类的内容
- 定期整理到相应文件夹

### `01-Inbox/待处理/`
- 存放需要进一步处理的内容
- 如：需要添加标签、需要AI分析等

## 剪藏流程

### 1. 快速剪藏
- 使用浏览器扩展一键剪藏
- 自动保存到默认文件夹
- 应用基础模板

### 2. 批量处理
- 每周集中处理剪藏内容
- 分类整理到相应文件夹
- 应用特定模板

### 3. 深度加工
- AI分析和生成摘要
- 添加个人笔记和思考
- 完善标签和元数据

## 模板使用

### 基础模板
- `网页剪藏模板.md` - 通用网页内容
- `技术文章模板.md` - 技术开发相关
- `学习资料模板.md` - 教育学习相关

### 模板选择策略
- 根据网站类型选择模板
- 根据内容性质选择模板
- 根据使用目的选择模板

## 搜索和索引

### 快速搜索
- 使用Ctrl+K进行全局搜索
- 使用标签筛选功能
- 使用Dataview进行复杂查询

### 索引文件
- 定期更新README文件
- 创建内容分类索引
- 建立标签关联索引

## 维护建议

### 日常维护
- 每天检查新的剪藏内容
- 及时添加合适的标签
- 删除重复或无用的内容

### 周期性维护
- 每周进行内容分类
- 每月整理标签系统
- 每季度清理过期内容

### 年度维护
- 复盘剪藏策略
- 更新模板系统
- 优化分类结构
