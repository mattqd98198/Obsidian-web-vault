## 🚀 快速使用技巧

### 1. Tag Group Manager快捷操作

#### 常用快捷键
- `Ctrl+K`: 打开搜索
- `Ctrl+N`: 新建标签组
- `Ctrl+E`: 进入编辑模式
- `Ctrl+F`: 搜索标签

#### 快速插入流程
1. 打开Tag Group Manager侧边栏
2. 选择对应标签组
3. 点击标签自动插入到当前笔记
4. 继续编辑其他内容

### 3. Dataview查询示例

#### 标签统计查询
```dataview
TABLE
  count(rows) as 使用次数,
  join(rows.tags, ", ") as 标签列表
FLATTEN tags as tag
GROUP BY tag
SORT 使用次数 DESC
LIMIT 20
```

#### 项目进度查询
```dataview
TABLE title, created, status
FROM "02-Projects"
WHERE contains(tags, "#工作项目")
  AND contains(tags, "#进行中")
SORT created DESC
```

#### 学习进度查询
```dataview
TABLE title, created, status
FROM "03-Resources"
WHERE contains(tags, "#学习资料")
  AND contains(tags, "#进行中")
SORT created ASC
```

## 📈 标签使用优化建议

### 定期维护
1. **每周清理**：删除无用的标签组合
2. **每月调整**：根据使用情况调整标签组
3. **季度回顾**：回顾标签使用效果
4. **年度规划**：规划新的标签分类

### 使用原则
1. **简洁明了**：标签名称简洁易懂
2. **层级清晰**：避免标签层级过深
3. **组合灵活**：支持多种标签组合
4. **易于扩展**：方便添加新标签
